# Conversor de real para dolar em TypeScript - PWEB
## Marcelo H. S. dos Santos

>*Objetivo:* Desenvolver uma aplicação funcional em Node.js utilizando TypeScript, implementando um sistema de menu iterativo e captura de dados do usuário.

>O projeto deve ser estruturado na pasta *src/* e conter os seguintes elementos obrigatórios:
>- *Tipagem Forte:* Todas as funções e variáveis de entrada devem possuir tipos explicitamente declarados (evite o uso de any).
>- *Módulo de Entrada:* Utilize a biblioteca promptSync para a interação com o terminal.
>- *Fluxo de Controle:* O programa deve rodar dentro de um laço de repetição (while ou do-while), permitindo múltiplas operações até que o usuário escolha a opção de saída.

> O tema escolhido foi o conversor de moedas. Suas funcionalidades mínimas são:
>1. BRL para USD; 
>2. USD para BRL; 
>3. Alterar Cotação; 
>0. Sair
